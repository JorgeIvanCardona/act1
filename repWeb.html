ETIQUETAS HTML
Etiqueta Función
!–…– Define un comentario
!doctype Define el tipo de docuemento
address Define la información de contacto del autor / propietario del documento
area Define un área dentro de un mapa de imagen
article Define un artículo
aside Define el contenido lateral del contenedor de una página
base Especifica la base donde se abrirán todas las URL del documento
bdi Aísla una parte del texto que puede tener un formato diferente del texto externo
body Define el cuerpo del documento
canvas Se usa para dibujar gráficos en pantalla
style Etiqueta de estilo usada para escribir CSS en línea.
meta Define los metadatos que no pueden ser definidos usando otro elemento HTML.
ESTILOS CSS
Estilos CSS Descripsión
Style Define estilos CSS
font-family: Define estilo de letra
color Define color de letra
background-color: Define color de fondo
table th, td { padding: ...} Define el alto de fila y recibe otros parámetros
fonts Define el tipo de letra
table, td { border: ...} Define el borde exterior y de celdas
Overflow Controla el comportamiento de los contenidos que no caben en su elemento contenedor
float Define la posición de la tabla en la ventana
Height Define el alto de tabla
width Define el ancho de tabla
